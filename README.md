# 🤖 AI Chatbot using Python

## 📌 Description
This is a simple AI chatbot built using Python and a language model API.  
It can generate human-like responses based on user input.

## 🚀 Features
- Conversational chatbot
- Uses AI/LLM for responses
- Simple and easy to understand

## 🛠️ Technologies Used
- Python
- Groq API / LLM
